# Countdown & Quotes App

**Modern countdown timer with auto-sliding inspirational quotes**

## Features
- Custom countdown timer (days, hours, minutes, seconds)
- 8 motivational quotes that auto-rotate every 3 seconds
- Pause/resume, next/previous quote controls
- Responsive design - works on all devices
- Smooth animations and gradient UI
- Visual status (running/stopped/finished)

## Quick Start
1. Save code as `index.html`
2. Open in browser
3. Pick future date/time
4. Click **Start Timer**

## Usage
- **Timer**: Set date → Start → Countdown begins
- **Quotes**: Auto-slide or use manual controls
- **Status**: Green=running, Red=finished, Gray=stopped
